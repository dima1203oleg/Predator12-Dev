❌ ТИПОВА ПОМИЛКА:
═══════════════════════════════════════════════════════════════════════

Ви робите:
1. npm run dev                    ✅ Сервер запускається
2. Бачите "ready in XXX ms"       ✅ Все добре
3. Натискаєте Ctrl+C              ❌ ПОМИЛКА! Сервер зупиняється
4. Намагаєтесь відкрити браузер   ❌ Не працює - сервер вимкнений!

🔄 РОБОТА З ДВОМА ТЕРМІНАЛАМИ:
═══════════════════════════════════════════════════════════════════════

Якщо потрібно виконати інші команди (curl, ls, etc.):

Варіант 1: VS Code
──────────────────
Terminal → New Terminal (Cmd+Shift+`)
Або клік на "+" біля терміналу

Варіант 2: iTerm/Terminal.app
─────────────────────────────
Cmd+T (нова вкладка)
або
Cmd+N (нове вікно)

Тепер маєте:
- Термінал 1: npm run dev (працює)
- Термінал 2: інші команди
