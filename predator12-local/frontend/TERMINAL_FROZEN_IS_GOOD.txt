🔍 ЯКЩО ЩОСЬ НЕ ТАК:

Сценарій А: Бачу білу сторінку або "Loading..."
───────────────────────────────────────────────────────────────────────
1. Натисніть F12 (або Cmd+Option+I на Mac)
2. Перейдіть на вкладку "Console"
3. Подивіться чи є червоні помилки
4. Скопіюйте текст помилок
5. Надішліть мені

Сценарій Б: Браузер каже "не може з'єднатись"
───────────────────────────────────────────────────────────────────────
Перевірте:
- Чи правильний URL: http://localhost:5092/
- Чи термінал все ще відкритий з npm run dev
- Спробуйте: http://127.0.0.1:5092/

Сценарій В: test.html працює, а головна сторінка ні
───────────────────────────────────────────────────────────────────────
1. Спробуйте: http://localhost:5092/test.html
2. Якщо бачите "Сервер працює" - сервер OK
3. Тоді проблема в React коді
4. Перевірте Console (F12) на головній сторінці

Сценарій Г: Частково працює
───────────────────────────────────────────────────────────────────────
Опишіть що саме бачите:
- Які елементи відображаються?
- Які ні?
- Є помилки в Console?

💡 КОРИСНА ІНФОРМАЦІЯ:

Як зрозуміти що сервер працює:
──────────────────────────────────────────────────────────────────────
✅ Термінал "завис" (не мигає курсор)
✅ Показано "Local: http://localhost:5092/"
✅ Можна відкрити http://localhost:5092/ в браузері

Як зрозуміти що сервер НЕ працює:
──────────────────────────────────────────────────────────────────────
❌ Курсор мигає в терміналі
❌ Можна вводити команди
❌ Браузер каже "Connection refused"

Що робити якщо потрібен інший термінал:
──────────────────────────────────────────────────────────────────────
Варіант 1: Відкрийте нову вкладку терміналу (Cmd+T)
Варіант 2: Відкрийте нове вікно терміналу (Cmd+N)
Варіант 3: У VS Code: Terminal → New Terminal

НЕ закривайте термінал з npm run dev!
